# Zarig.mn Style News Portal Demo

Live site: https://byambadorj251.github.io/lab6/

This project is a responsive Mongolian news portal demo with sections for politics, society, video news, columns, special projects, local updates, FAQ, sitemap links, and editorial contact information.

## SEO Files

- Homepage: `index.html`
- Robots: `robots.txt`
- Sitemap: `sitemap.xml`
- Manifest: `site.webmanifest`
- Social preview image: `og-cover.svg`

## Project Goal

The page is optimized for a static GitHub Pages deployment with canonical metadata, Open Graph tags, Twitter card tags, structured data, internal links, heading hierarchy, and crawlable supporting files.
